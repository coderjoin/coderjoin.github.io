Java 基础总结

一、基本数据类型
  • 8种基本数据类型：byte、short、int、long、float、double、boolean、char。
    二、String
  • 字符串的不可变性
  • intern方法在1.6和1.7的区别
  • StringBuffer（线程安全）和StringBuilder（线程不安全）的区别
    三、面向对象
  • 封装、继承、多态
  • is-a和has-a关系
  • override和overload关系
    四、异常
  • 继承Throwable接口，分为Exception和Error
  • Exception分为checked exception 和 unchecked exception
  • checked exception需要去catch，例如io exception
  • unchecked exception 例如 空指针、数组越界、参数不合法
    五、集合
  • Collections（静态帮助类）和Collection（接口）的区别
  • set和list是Collection的subinterfaces
  • 常用集合类的使用
  • ArrayList 和 LinkedList 和 Vector 的区别 、SynchronizedList 和 Vector 的区别、HashMap、HashTable、ConcurrentHashMap 区别
  • 特性和线程安全性问题
    六、IO
  • 基于字节、基于字符、基于网络（某种分类）
  • 装饰者模式
  • 同步、异步、阻塞、非阻塞、Linux 5 种 IO 模型
  • BIO、NIO 和 AIO 的区别、三种 IO 的用法与原理、netty
    七、Classloader
  • bootstarp classloader、extension classloader、application classloader
  • 双亲委托模型（从双亲检查加载确保安全）
    八、gc
  • 可达性分析（哪些可以作为起点）
  • 典型算法：mark-sweep、copying、mark-compact、generational collection
  • 7种垃圾收集器

以上就是自己顺着以前做过的Java language programming的ppt，脑海里回忆的Java 基础知识点，随便记录下来。高级部分，以后再说。。。。
